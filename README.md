POC of rewriting an existing [React app](https://github.com/doggen999/fluffy-octo-meme) to a Vue2 app.

## tl;dr

`yarn install` followed by

- `yarn dev` to start dev server...
- ...or `yarn build` to build a production bundle.

## Prerequisites

`Node v14.16.1`

`yarn v1.22.19`
